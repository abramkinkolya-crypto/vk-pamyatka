# Памятка ВК — ЛП по решению врачебной комиссии

Интерактивная веб-версия приложений 1.2, 1.3, 1.4 к приказу ГБУЗ «ГКБ им. В.П. Демихова ДЗМ» № 56 от 12.01.2026.

## Возможности
- Поиск по МНН и форме выпуска (с подсветкой совпадений)
- Фильтры: ВК+клинфармаколог / только ВК / без ВК / ЖНВЛП
- Статусные бейджи и указание приложения
- Адаптивная верстка (работает на телефоне)
- 131 позиция из приказа

## Публикация
Сайт развёрнут через GitHub Pages.

## Данные
Источник — приказ № 56 от 12.01.2026. Документ носит справочный характер; при расхождениях приоритет имеет оригинальный приказ.
