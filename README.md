# Формуляр ЛС 2026 · ГКБ им. В.П. Демихова ДЗМ

Интерактивная веб-версия приказа № 56 от 12.01.2026 — формуляр лекарственных средств на 2026 год.

## Включено
- **Приложение 1** — основной формуляр (со всеми позициями и признаком ЖНВЛП)
- **Приложение 1.1** — ЛС, подлежащие предметно-количественному учёту (ПКУ)
- **Приложение 1.2** — не ЖНВЛП, по решению ВК (с/без клинфармаколога)
- **Приложение 1.3** — ЖНВЛП, по решению ВК (с/без клинфармаколога)
- **Приложение 1.4** — антисептики и другие ЛС, не требующие ВК
- **Приложение 2** — правила работы с формуляром (встроено в страницу)

## Возможности
- Поиск по МНН и форме выпуска с подсветкой
- Фильтры: ВК+клинфармаколог / только ВК / любая ВК / ПКУ / ЖНВЛП / не ЖНВЛП / только формуляр
- Статусные бейджи и указание приложения
- Свернутая панель «Правила работы с формуляром»
- Адаптивная верстка (работает на телефоне)
- 439 позиций
